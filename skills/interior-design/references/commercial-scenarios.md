## 办公室

### 空间类型划分

| 类型 | 面积占比 | 说明 |
|------|---------|------|
| 开放办公区 | 50-70% | 主要工作区 |
| 会议/洽谈 | 10-15% | 大小会议室 + 洽谈区 |
| 管理层办公 | 10-15% | 独立办公室/半开放 |
| 公共服务 | 5-10% | 前台/茶水间/打印区/储物 |
| 设备用房 | 3-5% | 机房/强弱电间 |

### 动线设计

**三级动线分离：**

1. **客户动线**：入口 → 前台 → 洽谈/会议室 → 离开
   - 与员工区适当分隔
   - 前台可见、可达

2. **员工动线**：打卡 → 工位 → 茶水间/卫生间 → 会议室 → 离开
   - 最短路径
   - 避免穿越客户区

3. **后勤动线**：货运/快递 → 储物/打印区 → 各工位
   - 独立通道或利用次要动线

### 区域详解

**前台接待区：**
- 面积：≥6㎡，建议10-15㎡
- 前台桌：1.2-1.8m × 0.6m
- 等候座椅：2-4位，通道≥1.0m
- LOGO墙：距前台桌后0.8-1.2m
- 照明：200-300 lux（重点照明LOGO区）

**开放办公区：**
- 人均面积：≥4㎡，建议6-8㎡
- 工位排列：背靠背或面对面，过道≥0.9m
- 主通道：≥1.2m（消防≥1.4m）
- 靠窗优先给员工位
- 低隔断高度：1.1-1.2m（视觉遮挡+沟通便利）

**会议室：**
| 规格 | 面积 | 座位 | 桌子尺寸 | 屏幕距离 |
|------|-----|------|---------|---------|
| 小会议 | 8-10㎡ | 4-6人 | 1.2×0.6m - 1.8×0.9m | ≥1.5m |
| 中会议 | 15-20㎡ | 8-10人 | 2.4×1.0m - 3.0×1.2m | ≥2.0m |
| 大会议 | 30-40㎡ | 12-20人 | 4.0×1.2m + 投影 | ≥2.5m |
| 培训室 | ≥40㎡ | 20-40人 | 折叠桌/排座 | 讲台+投影 |

**独立办公室：**
- 进门预留≥60cm开门空间
- 班台背墙或背窗（避免背门）
- 书柜深度：30-40cm
- 会客沙发：2-3人位，茶几前通道≥60cm

**茶水间：**
- 面积：≥4㎡，建议6-10㎡
- 操作台长度：≥1.5m
- 冰箱位：600×600mm
- 微波炉/咖啡机位：≥60cm宽度
- 洗手池：宽≥45cm
- 就餐位：2-4人（可选）

### 照明与电源

- 工作区照度：300-500 lux
- 色温：4000-5000K（中性白）
- 显色指数：Ra≥80
- 每工位插座：≥4个（电脑+手机+预留）
- 地插：大会议室/培训室

### 空调与新风

- 办公区冷量：150-200W/㎡
- 新风量：≥30m³/h/人
- 风口避免直吹工位
- 独立办公室独立控制

## 展厅/Showroom

### 功能分区

| 区域 | 面积占比 | 功能 |
|------|---------|------|
| 入口形象区 | 10-15% | 第一印象、品牌展示 |
| 产品展示区 | 40-60% | 核心展示空间 |
| 洽谈区 | 10-15% | 商务洽谈、签约 |
| 多功能区 | 5-10% | 活动发布、培训 |
| 储物/办公 | 5-10% | 展品仓储、办公 |

### 尺寸规范

**通道宽度：**
- 主通道：≥1.5m（双向人流）
- 次要通道：≥1.0m
- 展品间距：≥0.8m（观展距离）

**展位规格：**
| 类型 | 尺寸 | 适用 |
|------|------|------|
| 标准展位 | 3×3m | 小型展品 |
| 中型展位 | 4×3m / 6×3m | 家具/厨卫 |
| 大型展位 | 定制 | 全屋样板间 |

**接待区：**
- 台桌：≥1.5m × 0.6m
- 等候座椅：2-4位
- 资料架距墙：≤30cm

### 灯光设计

**分级照明：**
| 区域 | 照度 | 色温 | 显色 |
|------|-----|------|-----|
| 入口形象 | 300-500 lux | 3000-4000K | Ra≥90 |
| 产品展示 | 500-1000 lux | 3000-4000K | Ra≥90 |
| 洽谈区 | 200-300 lux | 3000K | Ra≥80 |
| 通道 | 150-200 lux | 4000K | Ra≥80 |

**重点照明：**
- 射灯光束角：15°-36°
- 展品距墙：≤50cm
- 灯具投射角度：30°

### 材料耐久

- 地面：石材/大板砖（耐磨）
- 墙面：烤漆玻璃/金属板/石材（易清洁）
- 天花：石膏板+乳胶漆
- 踢脚：不锈钢/石材（耐撞）

## 餐饮空间

### 类型划分

| 类型 | 人均面积 | 厨房占比 | 特点 |
|------|---------|---------|------|
| 快餐/简餐 | 1.0-1.2㎡ | 25-30% | 高翻台率 |
| 自助餐厅 | 1.5-2.0㎡ | 30-40% | 取餐线+用餐区 |
| 正餐/中餐 | 1.5-2.0㎡ | 25-35% | 包房+大堂 |
| 西餐/咖啡 | 1.5-2.5㎡ | 15-25% | 情调优先 |
| 酒吧/清吧 | 1.5-2.0㎡ | 20-30% | 吧台+卡座 |

### 座位与餐桌

**通道宽度：**
- 主通道：≥1.2m
- 次要通道：≥0.9m
- 服务通道：≥0.6m（传菜）

**餐桌尺寸：**
| 类型 | 尺寸 | 座位数 |
|------|------|-------|
| 圆桌 Φ1.0m | 4人 | |
| 圆桌 Φ1.3m | 6人 | |
| 圆桌 Φ1.5m | 8-10人 | |
| 方桌 0.6×0.6m | 2人 | |
| 长桌 0.8×0.8m | 4人 | |
| 长桌 1.4×0.7m | 4-6人 | |

**座位间距：**
- 桌边到墙：≥60cm（靠墙）
- 桌间距：≥80cm（背靠背）
- 传菜通道：≥60cm

### 厨房设计

**功能分区：**
1. 原料入口/验收
2. 冷库/储藏
3. 粗加工（蔬果/肉类）
4. 热厨/烹饪
5. 冷菜/凉菜
6. 出菜口
7. 收餐/洗涤
8. 垃圾暂存

**尺寸规范：**
- 厨房通道：≥0.9m（单干），≥1.2m（双向）
- 操作台高度：85-90cm
- 设备间距：≥60cm
- 排烟罩高度：距灶台70-80cm

### 卫生合规

- 生熟分开（物理分隔或分区）
- 洗手池：入口处/粗加工区/冷菜区
- 地面排水坡度：≥2%
- 墙面瓷砖高度：≥1.5m（厨房），≥2.0m（灶台区）

## 零售店面

### 类型划分

| 类型 | 面积范围 | 特点 |
|------|---------|------|
| 便利店 | 50-150㎡ | 高密度货架 |
| 精品店 | 80-200㎡ | 产品体验+展示 |
| 专卖店 | 100-300㎡ | 品牌形象+销售 |
| 旗舰店 | ≥300㎡ | 全线产品+体验区 |

### 空间布局

**黄金分区：**

1. **入口区（10-15%）**：吸引视线、季节展示
2. **重点展示区（15-20%）**：新品/爆款
3. **常规陈列区（40-50%）**：主销商品
4. **收银区（5-10%）**：结账+冲动消费陈列
5. **仓储/办公（10-15%）**：货品储备

### 尺寸规范

**通道：**
- 主通道：≥1.2m（单人+侧身）
- 货架间通道：≥0.9m（顾客购物）
- 收银台前：≥1.5m（排队）

**货架规格：**
| 类型 | 高度 | 深度 | 说明 |
|------|------|------|------|
| 低架 | ≤1.2m | 40-50cm | 视线通透 |
| 中架 | 1.2-1.5m | 40-60cm | 主陈列 |
| 高架 | 1.5-1.8m | 50-60cm | 靠墙陈列 |
| 端架 | ≤1.5m | 40-50cm | 通道端头 |

**收银台：**
- 长度：1.2-1.8m
- 深度：0.6-0.8m
- 高度：0.85-0.95m
- 收银员通道：≥0.6m

### 照明

| 区域 | 照度 | 色温 |
|------|-----|------|
| 入口区 | 500-800 lux | 3500-4000K |
| 商品区 | 300-500 lux | 3500-4000K |
| 试衣区 | 300-400 lux | 3500-3800K |
| 收银区 | 300-400 lux | 4000K |

|------|---------|------|
| 接待大厅 | 15-20% | 迎宾/等候 |
| VIP包房 | 30-40% | 私密洽谈 |

| 餐饮区 | 10-15% | 简餐/茶歇 |

### 尺寸规范

**接待大厅：**
- 面积：≥30㎡
- 前台：1.5-2.0m × 0.6-0.8m
- 等候区：4-6位沙发
- 主通道：≥1.5m

**VIP包房：**
- 小包：15-20㎡（4-6人）
- 中包：25-35㎡（8-10人）
- 大包：≥40㎡（12人以上）
- 包房门：≥0.9m宽

**吧台区：**
- 吧台长度：≥2.0m
- 吧台深度：≥0.6m（操作面）
- 吧台高度：1.05-1.10m（站立）/ 0.75-0.85m（座台）
- 酒柜深度：35-45cm

## 消防与规范

### 安全出口

| 建筑面积 | 最少出口 | 单口最小宽度 |
|---------|---------|-------------|
| ≤200㎡ | 2个 | 0.9m |
| 200-500㎡ | 2个 | 1.2m |
| 500-1000㎡ | 3个 | 1.4m |
| ≥1000㎡ | 按320人/个计算 | 1.4m |

### 疏散距离

- 一、二级耐火建筑：双向疏散≤40m，单向≤20m
- 设有自动喷淋：可增加25%

### 防火分区

- 一级耐火：≤2500㎡（单层）
- 自动喷淋：可放大一倍
- 商业分隔：每个防火分区≤5000㎡

### 消防设施

- 灭火器：每75㎡一组（2具）
- 应急照明：地面照度≥1.0 lux
- 疏散指示：距地≤1.0m，间距≤20m
- 消防通道宽度：净宽≥1.4m，净高≥4.0m

## 灯光与声学

### 照度标准速查

| 空间 | 照度范围 | 色温 | 显色指数 |
|------|---------|------|---------|
| 办公区 | 300-500 lux | 4000-5000K | Ra≥80 |
| 会议室 | 300-500 lux | 3500-4500K | Ra≥80 |
| 展厅 | 300-1000 lux | 3000-4000K | Ra≥90 |
| 餐厅 | 150-300 lux | 2700-3500K | Ra≥80 |
| 零售 | 300-500 lux | 3500-4000K | Ra≥85 |
| 走道 | 100-150 lux | 4000K | Ra≥80 |

### 声学处理

**办公区：**
- 背景噪声：≤45dB
- 隔声量：≥35dB（会议室隔墙）
- 吸声材料：吊顶NRC≥0.6

**餐饮区：**
- 背景噪声：≤55dB（高峰）
- 包房隔声：≥40dB
- 厨房隔声：≥45dB

**展厅：**
- 背景噪声：≤40dB
- 混响时间：0.8-1.2s
- 材料NRC：≥0.5

- 1m = 3.28 ft
- 1 ft = 0.305m
