## 客厅

### 空间尺寸
- 面宽：≥3.6m（舒适≥4.2m）
- 进深：≥4m（舒适≥5m）
- 面积：≥15㎡（舒适≥25㎡）

### 家具规格
| 家具 | 常见尺寸 | 备注 |
|------|---------|------|
| 三人沙发 | 2.1-2.4m × 0.9-1.0m | 进深含靠背 |
| L型沙发 | 2.8-3.2m × 1.5-2.0m | 短边1.5-1.8m |
| 茶几（方） | 0.8-1.0m × 0.8-1.0m | |
| 茶几（圆） | Φ0.8-1.0m | |
| 茶几（长） | 1.2-1.4m × 0.6m | |
| 电视柜 | 1.8-3.0m × 0.35-0.4m | 深度35-40cm |
| 单人椅 | 0.7-0.85m × 0.7-0.85m | |

### 间距规范
- 沙发到电视：2.5-4.0m（视电视尺寸）
- 沙发到茶几：40-50cm
- 茶几到电视柜：≥80cm（通行）
- 主通道：≥90cm（舒适≥120cm）

### 电视尺寸与视距
| 电视尺寸 | 推荐视距 |
|---------|---------|
| 55寸 | 2.0-2.5m |
| 65寸 | 2.5-3.0m |
| 75寸 | 3.0-3.5m |
| 85寸 | 3.5-4.5m |

## 厨房

### 布局类型与最小尺寸
| 布局 | 最小面宽 | 操作台面 | 适用面积 |
|------|---------|---------|---------|
| 一字型 | ≥1.5m | ≥2.4m | 4-6㎡ |
| L型 | ≥1.8m | ≥3.0m | 5-8㎡ |
| U型 | ≥2.1m | ≥3.6m | 6-10㎡ |
| 双一字型 | ≥2.1m | ≥3.6m | 6-10㎡ |

### 尺寸规范
- 橱柜深度：地柜60cm，吊柜35cm
- 操作台高度：85-90cm（按使用者身高÷2+5cm）
- 吊柜底距台面：65-75cm
- 通道宽度：≥90cm（双人对面操作≥120cm）
- 水槽宽：≥60cm（单槽） / ≥80cm（双槽）
- 灶台两侧：≥30cm操作空间

### 动线三角
冰箱→水槽→灶台，三角周长3.6-6.6m为佳

## 办公室

### 空间尺寸
| 区域 | 最小面积 | 建议面积 |
|------|---------|---------|
| 开放办公区 | 人均≥4㎡ | 人均6-8㎡ |
| 独立办公室 | ≥10㎡ | 12-20㎡ |
| 会议室 | ≥8㎡（4人） | 15-30㎡ |
| 接待前台 | ≥6㎡ | 10-15㎡ |
| 茶水间 | ≥4㎡ | 6-10㎡ |

### 工位规格
| 类型 | 尺寸 | 适用场景 |
|------|------|----------|
| 标准工位 | 1.2×0.6m | 一般办公 |
| 舒适工位 | 1.4×0.7m | 设计/财务 |
| L型工位 | 1.6×1.2m | 设计师 |
| 高管大班台 | 1.8-2.4m × 0.9-1.0m | 独立办公室 |

### 通道与间距
- 主通道：≥1.2m（消防要求≥1.4m）
- 次要通道：≥0.9m
- 工位间背对背过道：≥0.9m
- 办公椅活动空间：椅后≥0.8m
- 文件柜前打开空间：≥0.6m

### 会议室
| 类型 | 面积 | 桌椅尺寸 |
|------|------|----------|
| 4人小会议 | 8-10㎡ | 桌1.2×0.6m |
| 6-8人会议 | 15-20㎡ | 桌2.4×1.0m |
| 10-12人会议 | 25-30㎡ | 桌3.0×1.2m |
| 大会议室 | ≥40㎡ | 固定座椅或阶梯 |

### 照明要求
- 办公区：300-500 lux
- 会议室：300-500 lux（可调光）
- 前台接待：200-300 lux
- 走道：100-150 lux

## 通用规范

### 门洞尺寸
| 位置 | 宽×高 |
|------|-------|
| 入户门 | 0.9-1.0m × 2.1m |
| 卧室门 | 0.8-0.9m × 2.1m |
| 厨房门 | 0.7-0.8m × 2.1m |
| 卫生间门 | 0.7m × 2.0m |
| 推拉门 | 1.4-1.8m × 2.1m |

### 窗户
- 窗台高：0.9m（卧室/客厅）/ 1.0m（厨卫）
- 窗顶距地：2.1-2.4m
- 落地窗：距地0-0.3m起

### 过道
- 主过道：≥100cm
- 次要过道：≥80cm
- 入户过道：≥120cm

### 层高
- 普通住宅：2.7-2.8m
- 舒适层高：2.9-3.0m
- 边吊后净高：≥2.4m（局部≥2.2m）
- 中央空调吊顶：下降25-30cm

### 开关插座
- 插座距地30cm（常规）/ 1.1m（床头/书桌）
- 开关距地1.3m
- 厨房台面插座距台面30cm
- 卫生间插座距地1.5m（防溅盒）

### 适老化补充
- 过道≥100cm（轮椅通行）
- 卫生间门宽≥80cm
- 马桶旁扶手：高70cm，距墙70cm
- 淋浴坐凳：高45cm
- 地面零高差
