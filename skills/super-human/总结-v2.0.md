## 使用方式

### 触发词示例

**开发模式**:
```
帮我做一个任务管理应用
```

**BMAD模式**:
```
用BMAD方法开发一个人力资源系统
```

**商业模式**:
```
新客户接入：做一个电商网站，预算5万
```

**Open Design模式**:
```
用Apple设计系统做一个SaaS落地页
```

**角色加载**:
```
作为安全工程师，请审查这段代码
```

✅ **开发+BMAD**: 既有严格TDD，又有敏捷迭代
✅ **商业+Open Design**: 快速生成专业原型，提升客户满意度
✅ **193个角色+72个设计系统**: 覆盖几乎所有专业场景
✅ **双轨记忆**: 快速上下文+深度存储，智能协同
