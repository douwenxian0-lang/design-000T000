# 全能超人 (Super Human) v2.0

**融合多个顶级技能的终极AI工作流系统** - 让AI真正理解你的需求，智能选择最佳工作流。

---

## 🎯 核心特性

### 1. **5合1智能融合**
- ✅ **开发模式** (superpowers) - TDD驱动，子代理协同
- ✅ **BMAD敏捷开发** - 6个专业角色，完整敏捷流程
- ✅ **商业管理模式** (agency) - 客户/项目/定价/交付
- ✅ **Open Design快速原型** - 31个设计模板，72个设计系统
- ✅ **193个专业角色** (agency-agents-zh) - 按需动态加载

### 2. **智能特性**
- 🧠 **双轨记忆系统** - 内置记忆 + 扩展记忆 (~/memory/)
- 🎨 **设计偏好学习** - 自动学习视觉偏好
- 🔄 **冲突自动优化** - 工作流程无缝衔接
- 🎛️ **模式自动识别** - 无需手动切换

## 🔍 疑难解答

### 问题1：模式识别错误

**症状**: 系统选择了错误的工作流模式

**修复**:
1. 明确指定模式：「用开发模式」、「用BMAD模式」、「用商业管理模式」
2. 检查触发词是否清晰
3. 重新训练模式识别系统

### 问题2：角色加载失败

**症状**: 无法加载请求的专业角色

**修复**:
1. 检查agency-agents-zh是否正确安装
2. 查看角色列表确认角色名称
3. 手动指定角色文件路径

### 问题3：记忆系统混乱

**症状**: 找不到存储的信息，或内置记忆与扩展记忆冲突

**修复**:
1. 检查 ~/memory/INDEX.md 是否最新
2. 验证内置记忆和扩展记忆的分离
3. 运行记忆系统诊断
